Fonctionnalités détaillées

💥 Modération ultra-rapide

Exemple : tape !ban @user pour bannir un utilisateur instantanément avec un message personnalisé.

🧠 IA anti-toxicité intelligente

Détecte automatiquement les propos offensants et prend des mesures (mute, warn) en fonction du contexte.

🔗 Anti-lien automatique

Bloque les messages contenant des liens non autorisés. Exemple : un message avec un lien est supprimé immédiatement.

📈 Système d’XP & niveaux

Chaque message rapporte de l'XP. Les utilisateurs peuvent voir leur progression avec !rank.

🎭 Commandes fun & troll

Amuse ta communauté avec !roast @user ou !8ball pour des réponses aléatoires drôles.

⚙️ Système de logs

Chaque action est loggée automatiquement dans un salon dédié. Exemple : logs de !kick dans #logs.
